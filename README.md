# api-server-sql
平台后端

## 快速上手
``` bash
git clone https://github.com/jgx020/pythonserverlesssql
cd pythonserverlesssql
git clone https://github.com/jgx020/pythonserverlesssql
pip install -r requirements.txt
```
然后就可以在`http://your-ip:5000/flask/`调试api
## 项目文件说明

```
.
├── bootstrap(阿里云serverless启动脚本)
├── index.py(flask应用默认配置脚本)
└── utils.py(flask蓝图功能)
```
